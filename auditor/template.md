## 📚 Glossary

| Term | Definition |
|---|---|
| **CPI** | Clock cycles consumed per retired instruction. Ideal ≈ 1.0 (in-order exec). |
| **AMAT** | Expected memory access time, weighted by cache-miss probability. |
| **P99** | 99th percentile latency — the "typical worst case" experienced by real workloads. |
| **NUMA Miss** | Memory access serviced by a remote NUMA node, incurring QPI bus penalty (~100 ns). |
| **IPS/W** | Instructions per second per watt — primary energy-efficiency KPI. |
